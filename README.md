## Extension Settings

The default keybinding of this plugin is ctrl+i. You can change it by adding

```
{
  "key": "ctrl+shift+i",
  "command": "extension.wrapTag",
  "when": "editorTextFocus"
}
```
on your vscode custom keybind setting file.
